Expense Tracker

React Application

This is a personal expense tracker. Made using React hooks and context API. It takes the amount inputted into the form and the name and lists it under the transaction history. If the number is positive then it will add to the balance and income, if the number is negative it will add to the expense but take away from the balance. This is perfect to use if you're trying to manage a budget and want to keep track of how much you are spending and how much you have available.
